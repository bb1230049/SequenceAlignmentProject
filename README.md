\# Nucleotide Sequence Alignment Tool



This project implements Smith-Waterman local alignment with affine gap penalty.



\## Features

\- Reads two FASTA files

\- Reads scoring parameters from file

\- Generates random DNA sequences

\- Outputs best local alignment

\- Saves result to file



\## Files

\- alignment\_tool.py

\- generate\_sequences.py

\- parameters.txt



\## Run Alignment

python alignment\_tool.py seq1.fasta seq2.fasta parameters.txt



\## Generate Test Sequences

python generate\_sequences.py 20 test
